# 简介

  • 首先，我们的网站是用 hexo + Github 进行搭建的，网站的域名是：https://team6940.github.io/

  • 在使用的过程中也可以参考一下官网的文档说明

# 需要的软件

  • 一个 Github 账户，注册方法自行百度。。。

安装 Hexo 相当简单，只需要先安装下列应用程序即可：

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • Git

如果您的电脑中已经安装上述必备程序，那么恭喜您！你可以直接前往 安装 Hexo 步骤。

如果您的电脑中尚未安装所需要的程序，请根据以下安装指示完成安装。

# 安装 Git

  • Windows：下载并安装 git.
  • Mac：使用 Homebrew 或 MacPorts 安装。
  • Linux（DEB/RPM-based）：从 NodeSource 安装。
  • 其它：使用相应的软件包管理器进行安装，可以参考由 Node.js 提供的 指导。

# 安装 Node.js

Node.js 为大多数平台提供了官方的 安装程序。

其它的安装方法：

  • Windows：通过 nvs（推荐）或者 nvm 安装、

Windows 安装说明：
当你点进链接时，会出现如下页面，然后点击图中框出的 “Release” 部分
< img src="https://pic.imgdb.cn/item/60d5d086844ef46bb24aab70.jpg" >

然后点击下图框选内容下载安装即可:

<img src="https://pic.imgdb.cn/item/60d5d2c4844ef46bb254af01.jpg">

  • Mac：使用 Homebrew 或 MacPorts 安装。
  • Linux（DEB/RPM-based）：从 NodeSource 安装。
  • 其它：使用相应的软件包管理器进行安装，可以参考由 Node.js 提供的 指导。

注意：

  • Windows 用户
    使用 Node.js 官方安装程序时，请确保勾选 Add to PATH 选项（默认已勾选）

# 安装 hexo

所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。
